# ORBiS – Service Support & Ticket Management System

## Overview
ORBiS is a service support and ticket management system developed for managing support requests, ticket workflows, and issue tracking.

## Features
- Ticket creation and management
- Ticket prioritization
- Dashboard and tracking
- Role-based workflows
- Support workflow management

## Technologies Used
- Django
- SQLite
- HTML/CSS

## Purpose
Built as a final semester academic project.
