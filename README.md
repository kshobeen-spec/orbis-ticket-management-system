# ORBiS  
### Support. Upgrade. Simplified.

A modern Service Support & Ticket Management System built using Django.

---

## Overview
ORBiS is a service support and ticket management system developed for managing support requests, ticket workflows, and issue tracking.

---

## Features

- User Authentication
- Ticket Creation & Tracking
- Admin Dashboard
- Internal Notes
- Ticket Conversations
- Service Management
- Help Center
- User Profiles
- Ticket Reply System
- Modern Dark UI

---

## Tech Stack

- Python
- Django
- SQLite
- HTML/CSS
- Bootstrap
- JavaScript

# Installation

```bash
git clone https://github.com/your-username/orbis-ticket-management-system.git
cd orbis-ticket-management-system

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
